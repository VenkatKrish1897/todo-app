# Todo App

A simple, lightweight todo application built with vanilla HTML, CSS, and JavaScript. No frameworks, no dependencies — just open the file and use it.

## Features

- Add todo items via input field or by pressing **Enter**
- Click a todo to mark it as **completed** (strikethrough)
- **Delete** individual todos
- **Dark / Light mode** toggle with persistence across sessions (localStorage)
- Responsive layout — works on mobile and desktop

## Getting Started

No installation required. Simply open `index.html` in any modern web browser:

```bash
# Clone the repository
git clone https://github.com/VenkatKrish1897/todo-app.git

# Open in browser
open todo-app/index.html
```

Or double-click `index.html` in your file manager.

## Usage

1. Type a task in the input field
2. Click **Add** or press **Enter** to add it to the list
3. Click on a todo item to toggle its completed state
4. Click **Delete** to remove an item
5. Use the **moon/sun** button in the top-right corner to switch between dark and light themes

## Tech Stack

- HTML5
- CSS3 (custom properties, CSS variables for theming)
- Vanilla JavaScript (no libraries or frameworks)

## Browser Support

Works in all modern browsers (Chrome, Firefox, Safari, Edge).
